# Sentiment Analysis of Medium App Reviews
Sentiment Analysis on Medium App
#📌 Overview
This project performs sentiment analysis on 62,633 real user reviews of the Medium application collected from the Google Play Store. The goal is to automatically classify each review as either positive or negative sentiment using Machine Learning techniques, helping businesses understand user feedback at scale.

#🎯 Problem Statement
Manually reading thousands of app reviews is time-consuming and inconsistent. This project automates the classification process using NLP and ML to extract meaningful insights from unstructured text data.
